# Grundprinzipien

* Abstimmungen:
    * sind geheim
    * Administratoren können von Ihnen ausgeschlossen werden (z.B. intern/vertraulich)
    * erfolgen innerhalb von Meetings, die jeweils einer Organisation (Tenant) zugeordnet werden
* Rollenkonzept:
  * Teilnehmer (nur für Abstimmungen)
  * Administratoren (bestimmen für eine Organisation die Tagesordnung und Abstimmungsstrukturen)
  * VoteEasy-Superadmin (erlaubt Verwaltung von Organisationen und Konfiguration der laufenden VoteEasy-Anwendung)
* Organisationen enthalten Beispieldaten, um einen einfachen Start für Nutzer zu ermöglichen
* datensparsame Teilnahme mit selbstdefiniertem Passwort ohne Angabe von E-Mails
* Selbstregistrierung neuer Organisationen möglich
